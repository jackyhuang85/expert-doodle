# Expert-Doodle

## Install
### Necessary Package
* opencv 
    * `$ conda install opencv`

* pyopencl 
    * `$ conda install -c conda-forge pyopencl`
    * Doc.: [PyOpenCL](https://documen.tician.de/pyopencl/)

* gputools 
    * `$ pip install gputools`
    * Github: [maweigert/gputools](https://github.com/maweigert/gputools)

## Run
`$ python main.py`
